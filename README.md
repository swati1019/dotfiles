# Arch Linux Dotfiles

My personal Arch Linux configuration focused on 
system automation and reliability.

## Setup
- OS: Arch Linux (ARM64, UTM VM on Apple M3)
- Shell: bash
- Editor: vim
- WM: i3 (config included)

## Structure
- `i3/` — i3 window manager config
- `i3status/` — status bar config  
- `scripts/` — automation shell scripts
- `.vimrc` — vim configuration

## Scripts
- `system-monitor.sh` — live CPU, RAM, disk usage monitor
